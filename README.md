# Student Performance Analyzer

## 📌 Descripción

Aplicación web desarrollada para analizar el rendimiento académico de estudiantes a partir de archivos Excel. El sistema permite procesar calificaciones, calcular promedios, generar estadísticas y visualizar resultados académicos mediante una interfaz web interactiva.

## 🛠 Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Bootstrap
- SheetJS
- Excel (.xlsx)
- Visual Studio Code

## ⚙️ Funcionamiento del sistema

La aplicación permite consultar y analizar información académica almacenada en archivos Excel.

El usuario puede seleccionar la información académica que desea consultar y el sistema procesa automáticamente los datos para generar resultados y estadísticas.

### Interfaz web

La interfaz fue desarrollada utilizando HTML, CSS y Bootstrap, permitiendo crear una estructura organizada con menús de navegación, tarjetas informativas y componentes visuales.

### Procesamiento de datos

JavaScript se encarga de leer, procesar y analizar la información de los archivos Excel mediante la librería SheetJS.

Los datos obtenidos son transformados para realizar cálculos y generar resultados académicos.

## ✨ Funcionalidades

- Lectura de archivos Excel (.xlsx).
- Consulta de estudiantes y calificaciones.
- Cálculo de promedios académicos.
- Análisis estadístico de resultados.
- Identificación de estudiantes con mejor rendimiento.
- Visualización organizada de información académica.

## 📊 Datos procesados

El sistema permite analizar:

- Promedio de Gestión Formativa.
- Promedio de Gestión Práctica.
- Promedio del Examen.
- Promedio general.
- Ranking de estudiantes según desempeño.
